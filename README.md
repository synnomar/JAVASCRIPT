# JAVASCRIPT Course

A beginner-level JavaScript course covering fundamental concepts from basic syntax to DOM manipulation.

## 📖 Documentation

For detailed information about each aula (lesson), **see [COURSE_DOCUMENTATION.md](COURSE_DOCUMENTATION.md)** for:
- Learning objectives for each lesson
- Code examples and explanations
- Key concepts and skills
- How to run each aula

## 📚 Quick Overview

| Aula | Topic | Learning Focus |
|------|-------|-----------------|
| **Aula 1** | Introduction & Hello World | Basic console.log |
| **Aula 2** | Variables & Loops | For/While loops, variables |
| **Aula 3** | HTML Integration | Browser console, script linking |
| **Aula 4** | Multiple Variables | String concatenation, let keyword |
| **Aula 5** | Data Types & Calculations | Multiple data types, calculations |
| **Aula 6** | User Input/Output | prompt(), alert(), type conversion |
| **Aula 7** | String Methods & DOM | String properties, DOM manipulation |

## 🚀 Getting Started

1. Open any aula folder
2. If there's an HTML file, open it in your browser
3. Press **F12** to open Developer Tools and check the Console
4. If there's only a JS file, run it with Node.js: `node filename.js`

All materials are in Portuguese.
Explore [COURSE_DOCUMENTATION.md](COURSE_DOCUMENTATION.md) for complete details!
